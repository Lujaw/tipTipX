TipTipX is a jQuery plugin based on Drew Wilson's TipTip which provides the ability to show CSS-styled tooltips. TipTipX provides extended functionality and somewhat improved tooltip positioning alghoritm.

Features

  • smart tooltip positioning
  • shows tooltips on hover, focus, click or triggered by javascript code
  • custom event hooks
  • works with Firefox 2/3, Opera 9, IE 6/7/8, Safari 3
  • open source (dual licensed under MIT and GPL licenses)

Examples

Dynamic positioning

Hover over the boxes to see tooltips. Resize your browser's window so that tooltips would not fit and hover again.

Static positioning

Resize your browser's window to see how tooltips reposition themselves.

Usage

Setup...

<script type="text/javascript" src="jquery.1.4.2.min.js"></script> <script type="text/javascript" src="jquery.tipTipX.js"></script>

Shown on hover on all elements with title attribute...

jQuery(function() { jQuery("[title]").tipTip(); });

Shown on focus...

jQuery(function() { jQuery(selector).tipTip({ activation : "focus" }); });

Shown with javascript code...

jQuery("#some_element").tipTipNow("#some_element's description");

Hide...

jQuery("#some_element").tipTipClose();

Change default options...

jQuery().tipTipDefaults({ delay : 0 });

Available options

  • suffix string
  • activation string event type, one of 'hover', 'focus' or 'click'
  • keepAlive boolean
  • maxWidth number
  • edgeOffset number
  • defaultPosition string
  • position string forces tooltip position
  • fadeIn number
  • fadeOut number
  • attribute string
  • content string
  • delay number
  • enter function
  • exit function

Known issues
