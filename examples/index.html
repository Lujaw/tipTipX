TipTipX is a jQuery plugin based on Drew Wilson's TipTip which provides the ability to show CSS-styled tooltips. TipTipX provides extended functionality and somewhat improved tooltip positioning algorithm.

Features

  • smart tooltip positioning
  • shows tooltips on hover, focus, click or triggered by javascript code
  • custom event hooks
  • tested with Firefox 2/3, Opera 10, IE 7/8, Safari 4/5
  • open source (dual licensed under MIT and GPL licenses)

Changing defaults

jQuery().tipTipDefaults({ delay : 0 });

Available options

  • tag string - by default only one tooltip can be shown at once, setting this option allows to show independent tooltips at the same time
  • activation string - type of event which triggers the tooltip, one of 'hover', 'focus' or 'click' ('hover' by default)
  • keepAlive boolean - if set to true tooltip will not fade out unless you hover over it and then hover off of it (false by default)
  • maxWidth number - maximal tooltip width ('600px' by default)
  • edgeOffset number - distance of tooltip from the element it is shown for (8 by default)
  • defaultPosition string - sets default position, one of 'top', 'bottom', 'left' or 'right' ('bottom' by default)
  • position string - forces tooltip position
  • fadeIn number - animation time in milliseconds (200 by default)
  • fadeOut number - animation time in milliseconds (200 by default)
  • attribute string - attribute which holds tooltip content ('title' by default)
  • content string - sets tooltip content explicitly
  • delay number - delay before tooltip is shown after hovering over an element (400 by default)
  • enter function - callback triggered when hovering over an element
  • exit function - callback triggered when hovering off an element

Known issues

Dynamic tooltip positioning does not work perfectly in all cases, but it is good enough for my uses. Anyways, patches are welcome.
